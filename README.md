# Sistema de Formulários e Workflow

Projeto desenvolvido no âmbito da unidade curricular/projeto, com organização do trabalho baseada em SCRUM e gestão através de GitHub Projects.

## Objetivo

Desenvolver um sistema que permita:

- criação e gestão de formulários por administradores
- publicação de formulários para públicos específicos
- submissão de processos
- associação de workflows aos formulários
- aprovação, rejeição, devolução para correção e consulta de estado dos processos
